## ✅ 新增功能

### 1️⃣ 全选功能

**位置**: 今日任务视图 → 批量操作菜单 → 第一项

**操作方式**:
```
1. 点击「批量操作」按钮
2. 选择「全选」选项
3. 所有当前筛选条件下的任务都会被选中
```

**功能特点**:
- ✅ 受筛选器影响（例如：筛选"待处理"时，只选中所有待处理任务）
- ✅ 带动画效果（0.2 秒缓动）
- ✅ 自动设置最后一个任务为上次选中项（支持后续 Shift+ 点击）

**使用场景**:
- 快速选中今日所有任务进行批量删除
- 一键选择所有待处理任务进行状态变更
- 配合筛选器快速选择特定类型的全部任务

---

### 2️⃣ 滑动多选功能（拖拽选择）

**位置**: 今日任务视图 → 任务列表

**操作方式**:
```
1. 点击「批量操作」按钮进入编辑模式
2. 在任意任务上按住鼠标左键
3. 不松开鼠标，上下拖动
4. 经过的所有任务都会被自动选中
```

**功能特点**:
- ✅ 只在编辑模式下启用（避免误触）
- ✅ 实时选择，无需点击
- ✅ 支持向上和向下拖动
- ✅ 自动记录起始位置
- ✅ 带动画效果（0.1 秒缓动）
- ✅ 不会取消已选中的任务（累加模式）

**技术实现**:
```swift
// 拖动选择手势
DragGesture(minimumDistance: 0, coordinateSpace: .local)
    .onChanged { value in
        handleDragGesture(taskIndex: taskIndex, location: value.location)
    }
    .onEnded { _ in
        endDragGesture()
    }
```

**使用场景**:
- 快速选择连续的一批任务（5-10 个）
- 不需要精确点击每个任务
- 类似 macOS Finder 的拖动选择体验

### 核心方法

#### 1. `selectAllTasks()`
```swift
/// 全选功能：选择所有当前筛选条件下的任务
private func selectAllTasks() {
    withAnimation(.easeInOut(duration: 0.2)) {
        selectedTaskIDs = Set(filteredTodayTasks.map { $0.id })
        if let lastTask = filteredTodayTasks.last {
            lastSelectedTaskID = lastTask.id
        }
    }
}
```

**关键点**:
- 基于 `filteredTodayTasks`（受筛选器影响）
- 使用 `Set` 去重
- 记录最后一个任务用于后续操作

---

#### 2. `handleDragGesture(taskIndex:location:)`
```swift
/// 处理拖动选择手势
private func handleDragGesture(taskIndex: Int, location: CGPoint) {
    // 只在编辑模式下启用拖动选择
    guard isEditingMode else { return }
    
    if !isDraggingToSelect {
        // 开始拖动，记录起始位置
        isDraggingToSelect = true
        dragStartIndex = taskIndex
        
        // 如果起始位置没有选中，先选中它
        let task = filteredTodayTasks[taskIndex]
        if !selectedTaskIDs.contains(task.id) {
            withAnimation(.easeInOut(duration: 0.1)) {
                selectedTaskIDs.insert(task.id)
            }
        }
    } else if let startIndex = dragStartIndex {
        // 正在拖动，选择起始位置和当前位置之间的所有任务
        let endIndex = taskIndex
        let actualStart = min(startIndex, endIndex)
        let actualEnd = max(startIndex, endIndex)
        
        withAnimation(.easeInOut(duration: 0.1)) {
            // 选择范围内的所有任务
            for i in actualStart...actualEnd {
                if i < filteredTodayTasks.count {
                    selectedTaskIDs.insert(filteredTodayTasks[i].id)
                }
            }
        }
    }
}
```

**关键点**:
- 只在编辑模式下工作（防止误触）
- 累加模式（不会取消已有选择）
- 边界检查（防止数组越界）
- 动画平滑过渡

---

#### 3. `endDragGesture()`
```swift
/// 结束拖动选择手势
private func endDragGesture() {
    isDraggingToSelect = false
    dragStartIndex = nil
}
```

**作用**: 清理状态，准备下一次拖动

---

### 状态变量

```swift
@State private var isDraggingToSelect = false  // 是否正在拖动选择
@State private var dragStartIndex: Int?       // 拖动起始位置索引
```

## 💡 设计原理

### 为什么这样设计？

**1. 全选的便利性**
- 用户经常需要一次性选择所有任务
- 逐个点击效率低下
- 一键全选大幅提升效率

**2. 拖动选择的直觉性**
- macOS Finder 的经典交互
- 符合用户习惯
- 比 Shift+ 点击更直观

**3. 编辑模式限制**
- 防止日常使用时误触
- 明确的操作意图
- 区分浏览和操作模式

**4. 累加而非替换**
- 允许分批次选择
- 更灵活的选择策略
- 不会意外取消选择
