## 📋 概述

本次迭代为 PlanCheck 应用添加了两大核心功能：**全局字体缩放**和**窗口尺寸记忆**，显著提升用户体验和无障碍访问能力。

---

## ✨ 新增功能

### 1. 全局字体缩放系统

#### 快捷键操作
| 快捷键 | 功能 | 效果 |
|--------|------|------|
| **⌘ + =** | 放大字体 | +2pt（最大 32pt） |
| **⌘ + -** | 缩小字体 | -2pt（最小 10pt） |
| **⌘ + 0** | 重置字体 | 回到默认 14pt |

#### 技术实现
- **FontManager 单例类**：管理全局字体大小状态
- **@Published 响应式更新**：字体变化即时生效
- **UserDefaults 持久化**：自动保存用户偏好
- **Environment 环境值传递**：全应用统一缩放
- **智能上下文感知**：文本编辑时仍可缩放

#### 核心特性
- ✅ 12 级可调（10, 12, 14, ..., 32pt）
- ✅ 所有文本等比缩放，保持层次关系
- ✅ 自动记忆，下次启动恢复
- ✅ 无需重启应用，实时生效

### 2. 窗口尺寸管理

#### 功能特性
- **窗口尺寸持久化**：自动保存关闭时的窗口大小
- **启动时自动恢复**：记住用户的窗口偏好
- **平滑动画**：尺寸变化带有流畅过渡效果

#### 技术实现
- **WindowManager 单例类**：管理窗口尺寸
- **NSWindow setFrame**：带动画调整尺寸
- **UserDefaults 存储**：保存宽高数据
- **延迟恢复机制**：避免启动闪烁

## 📁 新增文件

### 1. FontManager.swift (92 行)

class FontManager: ObservableObject {
    static let shared = FontManager()
    
    @Published var currentFontSize: CGFloat
    
    func zoomIn()      // 放大字体
    func zoomOut()     // 缩小字体
    func resetZoom()   // 重置
    func scaledSize()  // 计算相对大小
}

## 🚀 使用指南

### 快速开始
1. 打开 PlanCheck 应用
2. 按 `Command + =` 放大字体
3. 按 `Command + -` 缩小字体
4. 按 `Command + 0` 重置到标准大小
5. 关闭窗口，下次打开自动保持设置

### 推荐配置
- **标准视力**：14pt（默认）
- **轻度老花**：16-18pt
- **中度老花**：20-24pt
- **严重视力障碍**：26-32pt
