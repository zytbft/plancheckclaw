### 功能 2：全部任务 - 添加到今日任务

**实现位置**: 
- `AllTasksView.swift` - 全部任务视图

**功能描述**:
- 在批量操作菜单中添加"添加到今日任务"选项
- 将选中的多个任务批量复制到"我的一天"列表
- 自动过滤已加入的任务，避免重复
- 操作完成后自动退出编辑模式并清空选择

**代码实现**:
```swift
/// 批量添加到今日任务
private func addSelectedToToday() {
    var addedCount = 0
    for taskID in selectedTaskIDs {
        if let task = allTasks.first(where: { $0.id == taskID }) {
            // 如果任务还未加入"我的一天"，则加入
            if !task.isInMyDay {
                taskStore.addToMyDay(taskID: taskID)
                addedCount += 1
            }
        }
    }
    
    // 显示成功提示
    if addedCount > 0 {
        withAnimation {
            isEditingMode = false
            selectedTaskIDs.removeAll()
        }
        print("已添加 \(addedCount) 个任务到今日任务")
    }
}
```

### 功能 3：今日任务 - 从今日任务移除

**实现位置**: 
- `TodayView.swift` - 今日任务视图

**功能描述**:
- 在批量操作菜单中添加"从今日任务移除"选项
- 将选中的多个任务从"我的一天"中移除（但不删除任务本身）
- 被移除的任务会回到全部任务列表，并根据排序规则移动到顶部
- 操作完成后自动退出编辑模式并清空选择

**代码实现**:
```swift
/// 批量从今日任务移除
private func removeSelectedFromMyDay() {
    var removedCount = 0
    for taskID in selectedTaskIDs {
        if let task = todayTasks.first(where: { $0.id == taskID }) {
            // 只处理已加入"我的一天"的任务
            if task.isInMyDay {
                removeFromMyDayAndUpdateSort(taskID: taskID)
                removedCount += 1
            }
        }
    }
    
    // 显示成功提示
    if removedCount > 0 {
        withAnimation {
            isEditingMode = false
            selectedTaskIDs.removeAll()
        }
        print("已从今日任务移除 \(removedCount) 个任务")
    }
}
```

## 🔧 技术细节

### 1. 全选逻辑

**关键点**:
- 使用 `filteredTodayTasks` 或 `filteredTasks` 确保只选中当前筛选条件下的任务
- 使用 `withAnimation` 包裹以提供平滑的动画效果
- 通过 `Set` 数据结构自动去重

**示例**:
```swift
private func selectAll() {
    withAnimation {
        // 将过滤后的任务 ID 数组转换为 Set
        selectedTaskIDs = Set(filteredTodayTasks.map { $0.id })
    }
}
```

### 2. 添加到今日任务

**业务逻辑**:
1. 遍历所有选中的任务 ID
2. 检查任务是否已在"我的一天"中
3. 如果不在，调用 `taskStore.addToMyDay(taskID:)` 加入
4. 统计成功加入的数量
5. 如果有任务被加入，退出编辑模式并清空选择

**智能过滤**:
```swift
if !task.isInMyDay {
    taskStore.addToMyDay(taskID: taskID)
    addedCount += 1
}
```

### 3. 从今日任务移除

**业务逻辑**:
1. 遍历所有选中的任务 ID
2. 检查任务是否在"我的一天"中
3. 如果在，调用 `removeFromMyDayAndUpdateSort(taskID:)` 移除
4. 统计成功移除的数量
5. 如果有任务被移除，退出编辑模式并清空选择

**复用现有方法**:
```swift
// 复用已有的单个移除方法
removeFromMyDayAndUpdateSort(taskID: taskID)
```

## ⚠️ 注意事项

### 1. 数据一致性

- **添加到今日任务**: 会自动过滤已加入的任务，避免重复
- **从今日任务移除**: 只处理已在"我的一天"中的任务

### 2. 性能考虑

- 全选操作使用 `map` 和 `Set` 转换，时间复杂度 O(n)
- 对于大量任务（1000+），建议使用筛选器缩小范围

### 3. 用户习惯

- 操作完成后自动退出编辑模式，避免误操作
- 清空选择状态，防止下次批量操作时混淆
